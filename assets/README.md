# Assets Directory

This directory is intended for project-wide assets like logos, diagrams, and promotional materials.

## Suggested Contents

### Logo
- `logo.png` - Full-size app logo (transparent background)
- `logo_banner.png` - Wide banner format for README header
- `icon.png` - Square icon version

You can export these from the app's mipmap resources:
```
android/app/src/main/res/mipmap-xxxhdpi/mindbricks_logo.webp
```

### Architecture Diagrams
- `architecture_diagram.png` - System architecture overview
- `database_schema.png` - ER diagram for Room database

### Other Assets
- Feature comparison tables
- Flowcharts for complex logic
- UX wireframes or mockups

---

**Note**: The app icon already exists in `android/app/src/main/res/mipmap-*/` directories.
